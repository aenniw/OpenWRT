## Setup environment
```
sudo apt-get update
sudo apt-get install cmake make
```

## Build
```
cmake ./CMakeLists.txt
for PROJECT in $(ls -d ./*/CMakeLists.txt); do \
    cd ${PROJECT%*/*} && make; cd ..; \
done
```
