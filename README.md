## Setup environment

```bash
sudo apt-get update
sudo apt-get install cmake make wget tar
```

## Build

To change toolchain add architecture to CMake like so -Dar71xx=true or -Dipq40xx=true

```bash
    ./make ar70xx
    ./make ipq40xx
```
